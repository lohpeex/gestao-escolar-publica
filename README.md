# Sistema de Gestão Escolar Pública

## Descrição
Este projeto apresenta a proposta de um sistema de gestão escolar voltado à educação pública, com foco na organização de dados e apoio à gestão administrativa.

## Objetivo
Criar uma estrutura de sistema que auxilie no controle de alunos, frequência e comunicação com responsáveis.

## Funcionalidades propostas
- Cadastro de alunos
- Controle de frequência
- Registro de atividades
- Comunicação com responsáveis
- Geração de relatórios

## Tecnologias (proposta)
- Java ou PHP
- MySQL
- Interface web

## Contexto
O projeto foi inspirado na realidade da educação pública, considerando desafios na organização e integração de dados escolares.
